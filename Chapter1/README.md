# 1. 기초 필수 수학

두 벡터의 뺄셈은 A 벡터의 머리에서 B 벡터의 머리로 이동하는 벡터를 구하는 계산

벡터의 크기(길이)는

sqrt(a^2 + b^2 + c^2) 이다.

## 두 벡터의 값을 아는 상황에서 각도 구하기

A = (1,2,3)
B = (-4,0,-1)

내적이
||A|| ||B|| cosθ
이므로,
||A|| = sqrt(14)
||B|| = sqrt(17)
cosθ = -7 / sqrt(14) * sqrt(17)
이 된다.
